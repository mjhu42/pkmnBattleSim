This program an interactive Pokémon battle simulator. You can pick the 6 Pokémon that you want for your team, as well as their moves and EV distribution, and then play against the computer in a battle on either an easy or difficult level.

## SETUP / INSTALLATION
This program requires Pygame to run. If you do not have Pygame installed, it is retrievable from pip (more information here: https://www.pygame.org/wiki/GettingStarted). To start up the actual program, run PokemonBattleSimulator.py.

## GAMEPLAY
Once you enter the select screen, click on one of the boxes on the right sidebar to add a Pokémon to your party. You can then enter its name into a text field, its level, its moves, et cetera. Make sure that you hit "enter" after you've typed in whatever input, or else the information will not be saved. Then, select a CPU leve, and click play!

## FONTS
It is strongly recommended that the following fonts be installed, if they have not been already:
   - Bahnschrift (typically included in the system as a font already)
   - Calibri (typically included in the system as a font already)
   - Futura (download available: https://www.wfonts.com/font/futura)
   - Gameplay (file included in the Fonts folder)
   - Pokemon Solid (file included in the Fonts folder)
   - Spantarna (demo) file included in the Fonts folder)
